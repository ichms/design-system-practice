# CLAUDE.md

이 프로젝트는 디자인 시스템 비교 분석을 위한 학습/리서치 레포다.

---

## 규칙

### 질문 마커 (`**Q.`)

마크다운 파일 안에서 `**Q.` 로 시작하는 줄을 발견하면, 해당 질문 바로 아래에 답변을 작성한다.

- 질문 원문은 그대로 유지한다.
- 답변은 간결하고 핵심만 담는다. 필요한 경우 예시 코드를 포함한다.
- 문답 전체를 아래 HTML 블록 형식으로 감싼다. 한눈에 문답임을 알아볼 수 있어야 한다.

**포맷 (반드시 이 형식 사용):**

- 🔶 = 질문 (주홍색 느낌)
- 🟢 = 답변 시작 (초록색 느낌)
- 전체를 blockquote(`>`)로 감싸 본문과 시각적으로 구분

```md
> 🔶 **Q. 질문 내용**
>
> 🟢 답변 첫 문장.
>
> 이어지는 내용은 `>` 유지하며 계속 작성.
> 목록도 blockquote 안에서 그대로 사용 가능:
>
> - 항목 1
> - 항목 2
```
